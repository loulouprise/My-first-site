Le Web et ses infrastructure

Dans le web il y a aussi le Cloud les Base de données big data, SQL, cookie (aspect informationnel), RGPD...

Où son stocker toute les donner du web ?

Les donnés du web comme les site les fichier les image sont tous stocker dans des data center aussi appelée base de données avec a l'intérieur des serveurs.
C'est pour sa que nous pouvons voir des problèmes avec site quand le data center de Paris nommé Paris Digital Park qui possède plus de 7 hectares de serveurs. Ce que les gens apelle le cloud ce sont enfaite des serveur qui stock vos information. Par exemple Onedrive est un cloud qui stock vos information et vos fichier dans un serveur qui vous est dédié qui permet de ne pas perdre vos information si jamais vous les aviez stocker sur votre appareil.

Qu'est ce que le SQL ?

Le SQL est le language infomatique que votre ordianteur envoie en requête sous forme de paquet au serveur. Une fois que le serveur a trouver ce que vous rechercher alors a ce moment la le serveur va vous renvoyer une requête en SQL qui serra automatiquement traduit par votre ordinateur.

Vous acceptez tous le temps les coukies mais avez vous réellement ce que c'est ?

Les coukies sont de petit fichier texte enregistrer sur votre appareil et que va servir a votre appareil a vous faire des publicité plus ciblé. Le RGPD, qu'est-ce que c'est ?

Le RGPD x'est Le règlement général de protection des données (RGPD) est un texte réglementaire européen qui encadre le traitement des données de manière égalitaire sur tout le territoire de l'Union européenne (UE).
